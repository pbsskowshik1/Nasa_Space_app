## Nasa_Space_app
This website is really cool because it teaches kids about space weather. It talks about things like flares and solar wind. The website uses characters and stories to make learning fun. It also has quizzes to help kids see what they have learned about space weather. The website is a place for young learners to learn about space weather and things, like solar flares and solar wind.

go to cd server and in the next line node index.js run it in terminal.
open cmd and then run this command  in the client file npm run dev.

Note : For this u need to have node js installed.

## Features
- Interactive, character-driven storytelling that explains space weather phenomena (solar flares, CMEs, solar wind)
- JSON-driven branching narrative engine for dynamic story paths
- Adaptive quizzes to reinforce learning
- Real-world scenarios showing impact on GPS, satellites, aviation, and power grids
- Designed for early learners, simplifying complex astrophysics concepts

## Tech Stack
- HTML, CSS, JavaScript — structure, styling, and interactivity
- JSON — story/quiz content and branching logic

## What I Learned
Building this project taught me how to structure branching narratives in JSON so the story logic stayed manageable as it grew. Translating dense astrophysics concepts (like coronal mass ejections) into simple, relatable scenarios for young learners was the biggest challenge — it pushed me to think about how to simplify technical material without losing accuracy. I also learned to think about real-world impact early, aligning the project with UN SDG 4 and SDG 13.
